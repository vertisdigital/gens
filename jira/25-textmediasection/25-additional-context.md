# Authoring Requirements:
create a new block with the following specifications
        1. a parent block with the following specifications
            a. A select component named "layout" with the following options
               i. Media First
               ii. Text Section First
        2. The parent block can have the following children blocks
            a. image block
            b. existing text section block

# Rendering Requirements:
1. based on the layout selected, the image block should be rendered first or the text section block should be rendered first
2. use already existing text section block as is
3. refer the images provided with the jira ticket
