# Rendering Requirements:
    1. Layout & Structure:
        - Follow bootstrap grid system. All classes must pick from styles.css
        Desktop:
            - Responsive grid layout using container-xl, row, col-xl-3 classes. 
        Tablet
            - Responsive grid layout using container-md, row, col-md-3 classes.
        Mobile
            - - Responsive grid layout using container-sm, row, col-sm-4 classes.
    
    2. Typography:
        - Use system font stack for menu items
        - Column headers should be bold
        - Footer text should be smaller than main content
    
    3. Styling:
        - Light text on dark background (brown/bronze color scheme)
        - Subtle hover effects on links
        - Social media icons with hover states
        - Proper spacing between menu items
    
    4. Accessibility:
        - Semantic HTML structure using nav, footer elements
        - ARIA labels for navigation sections
        - Proper heading hierarchy
        - Keyboard navigation support
        - High contrast ratios for text
    
    5. Technical Requirements:
        - Use data attributes for dynamic content (data-aue-label)
        - Implement proper HTML5 semantic structure
        - Mobile-first responsive design
        - Support for dynamic menu items
        - Lazy loading for images

# Brand Guidelines:
    - Color Scheme: Bronze/Brown (#8B7355) for background
    - Typography: System font stack
    - Logo: Maintain proper clear space
    - Social Icons: Consistent size and spacing
