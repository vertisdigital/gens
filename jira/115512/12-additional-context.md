# Authoring Requirements:
    ## This will have following fields
    1.    A text Heading
    2.    A rich text Description
    3.    A image

# Rendering Requirements:
    Use UL LI Syntax
    Text and Description should be in a single column on the left side of the image and wrapped in single Block. Description should be under Heading
    Image should be on the right side of the text and description
    Pick the font, color, layout and other styling as per the image provided
    Use data attributes for the elements and items which will come from snippet such as data-aue-label="Title" etc.
